# Personal portfolio
This is a [Next.js](https://nextjs.org/) project with [Typescript](https://www.typescriptlang.org/) and [Tailwind CSS](https://tailwindcss.com/).


### Run the development server:

```bash
npm run dev
# or
yarn dev
```
